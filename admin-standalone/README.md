# Standalone Admin App

This folder is a root-ready standalone version of the newer admin pages.

Deploy this folder as the root of a separate site, for example:
- `admin.yogaunnati.app`

Suggested GitHub Pages shape:
- separate repo
- copy the contents of this folder to that repo root
- set `CNAME` to `admin.yogaunnati.app`

Uses the same Supabase backend as the main app.
